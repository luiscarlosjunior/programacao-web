Conceitos de Acesso ao DOM

document.getElementById()

Acessa um elemento pelo seu ID.

Exemplo: document.getElementById('meuId')

document.getElementsByClassName()

Acessa elementos por classe.

Exemplo: document.getElementsByClassName('minhaClasse')

document.getElementsByTagName()

Acessa elementos por tag.

Exemplo: document.getElementsByTagName('div')

document.querySelector()

Acessa o primeiro elemento que corresponde a um seletor CSS.

Exemplo: document.querySelector('.minhaClasse')

document.querySelectorAll()

Acessa todos os elementos que correspondem a um seletor CSS.

Exemplo: document.querySelectorAll('div.minhaClasse')
