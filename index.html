Explicação da Tag DIV

O que é a tag DIV?

A tag <div> é um elemento de bloco genérico usado para agrupar outros elementos HTML e aplicar estilos ou scripts a esses grupos.

Exemplo de Uso

Veja abaixo um exemplo simples de uso da tag <div>:

<div class="container">
    <h2>Título</h2>
    <p>Este é um parágrafo dentro de uma div.</p>
</div>

Por que usar a tag DIV?

A tag <div> é útil para:

  • Organizar o layout da página
  • Agrupar elementos para aplicar estilos CSS
  • Manipular grupos de elementos com JavaScript
